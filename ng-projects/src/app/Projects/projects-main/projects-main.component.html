Calculator

A quick web app I built using bootstrap and TypeScript. First I added two variables into my component class: public input:string = ' '; public output: number = 0; originally I made both variables public as I was retrieving them from my app component. I then created several functions. onSubmit() adds value to input. onClear() resets the values of the variables. onRemove uses substring() and .length to remove the last index from input (-1) onEquals has a variable called result which calls an anon function with the arguments ('return ' + this.input) and then prints the answer with this.output = result();

Calculator

A web app built for a little bit of fun. Users ask a question to the magic ball and it responds with an answer from a list of random answers. This app was built inside of Angular using bootstrap and css for the design and Typescript for the logic. I created 2 variables within my component class namely: eightBall: string = '' and randomArr: Array<string> = ['answers here'] I then created a function called "onRandom" and targeted "eightBall" to populate the answer within the array using its length. The Answers were generated randomly using Math.random() and was rounded up to the nearest full index number with Math.floor(). I finally returned this.eightball
